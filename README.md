# Claude Token Efficiency Evaluator 🚀

**Claude Token Efficiency Evaluator** is a Julia-based NLP tool I built using Anthropic’s Claude API. It analyzes NLP prompts, clearly providing insights on token efficiency, redundancy, and potential cost savings.

---

## 🎯 How It Works:

- Takes a natural language prompt.
- Sends the prompt to Claude API to evaluate clearly:
  - Token count.
  - Redundant phrases.
  - Suggested improved prompt.
  - Estimated Claude API cost.
- Displays the analysis in a clear, formatted output.
